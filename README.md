WALLTY – RUTES I PUNTS
Camina i guanya

Wallty és una aplicació web que combina activitat física amb recompenses digitals. L’objectiu és incentivar els usuaris a caminar i completar rutes mentre acumulen Walltycoins, una moneda virtual utilitzable dins la plataforma.

FUNCIONALITATS PRINCIPALS

Sistema d’autenticació (email/password + Google)
Verificació de correu electrònic
Recuperació de contrasenya
Sistema de rutes (explorar i completar)
Sistema de recompenses amb Walltycoins
Perfil d’usuari amb seguiment de progrés
Persistència de dades amb Firebase

IDEA DEL PROJECTE

Wallty neix amb l’objectiu de motivar les persones a portar un estil de vida actiu a través de la gamificació. L’usuari no només camina, sinó que progressa, acumula recompenses i té objectius clars.

SISTEMA D’AUTENTICACIÓ

L’usuari pot registrar-se, iniciar sessió, verificar el correu electrònic, recuperar la contrasenya i accedir amb Google.

FUNCIONAMENT DEL SISTEMA DE RUTES

1. L’usuari selecciona una ruta
2. Completa la ruta
3. El sistema registra la finalització
4. Rep una recompensa en Walltycoins
5. El progrés es guarda a Firebase

PROBLEMES ACTUALS

Les rutes completades no sempre es guarden correctament
Falta optimització dels estats de l’aplicació
Interfície d’usuari millorable
